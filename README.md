Userstock Valuation Project
A full-stack stock valuation and trading simulator built with Node.js, MongoDB, Express, and React. The platform enables users to execute buy/sell trades, manage stock portfolios, monitor real-time holdings and positions, and analyze performance, all with automatic backend state updates.


Features

User-friendly web UI for searching stocks, portfolio management, and viewing analytics.
Place buy & sell orders; backend auto-updates Holdings and Positions with each order.
Live P&L, average buy tracking, real-time price updates, trading history, and position management.
Modular API endpoints for integrations and extensions.
Clean codebase with separation of frontend and backend logic.

Tech Stack
Frontend: React, Axios, Material UI (MUI)
Backend: Node.js, Express, Mongoose (MongoDB ODM)
Database: MongoDB (local/cloud)
Other: dotenv, cors, body-parser

Start Backend:  npm start
start fronend : npm start
